# Las librerías `algorithm` y `math`

Las librerías `algorithm` y `math` son dos de las librerías estándar de C++ que proporcionan funciones y algoritmos
útiles para realizar operaciones matemáticas y de manipulación de datos. En este artículo, exploraremos las
características y funcionalidades de estas dos librerías, así como algunos ejemplos de su uso en la programación en C++.

## `algorithm`

La librería `algorithm` proporciona una serie de funciones y algoritmos para realizar operaciones comunes en
contenedores de datos, como vectores, listas y arreglos. Algunas de las funciones más utilizadas de esta librería
son:

* `std::sort()`: Ordena los elementos de un contenedor en orden ascendente.
* `std::reverse()`: Invierte el orden de los elementos de un contenedor.
* `std::find()`: Busca un elemento en un contenedor y devuelve un iterador al elemento encontrado.
* `std::count()`: Cuenta el número de ocurrencias de un elemento en un contenedor.
* `std::accumulate()`: Calcula la suma de los elementos de un contenedor.
* `std::transform()`: Aplica una función a cada elemento de un contenedor y guarda los resultados en otro contenedor.
* `std::copy()`: Copia los elementos de un contenedor en otro contenedor.
* `std::unique()`: Elimina elementos duplicados de un contenedor.
* `std::min_element()`: Encuentra el elemento más pequeño en un contenedor.
* `std::max_element()`: Encuentra el elemento más grande en un contenedor.
* `std::binary_search()`: Busca un elemento en un contenedor ordenado utilizando búsqueda binaria.
* `std::merge()`: Combina dos contenedores ordenados en un solo contenedor ordenado.
* `std::partition()`: Divide un contenedor en dos partes, una que cumple cierta condición y otra que no.
* `std::rotate()`: Rota los elementos de un contenedor.
* `std::shuffle()`: Mezcla los elementos de un contenedor de forma aleatoria.

Estas funciones y algoritmos son muy útiles para realizar operaciones de búsqueda, ordenación, transformación y
manipulación de datos en contenedores de datos en C++.

## `math`

La librería `math` proporciona una serie de funciones matemáticas para realizar operaciones comunes en números y
datos numéricos. Algunas de las funciones más utilizadas de esta librería son:

* `std::abs()`: Calcula el valor absoluto de un número.
* `std::sqrt()`: Calcula la raíz cuadrada de un número.
* `std::pow()`: Calcula la potencia de un número.
* `std::sin()`, `std::cos()`, `std::tan()`: Calculan las funciones trigonométricas seno, coseno y tangente de un ángulo.
* `std::asin()`, `std::acos()`, `std::atan()`: Calculan las funciones trigonométricas inversas del seno, coseno y
  tangente.
* `std::log()`, `std::log10()`, `std::exp()`: Calculan el logaritmo natural, el logaritmo en base 10 y la función
  exponencial de un número.
* `std::ceil()`, `std::floor()`, `std::round()`: Redondean un número hacia arriba, hacia abajo y al entero más
  cercano, respectivamente.
* `std::min()`, `std::max()`: Devuelven el mínimo y el máximo de dos números.
* `std::fmod()`: Calcula el resto de la división de dos números.
* `std::isnan()`, `std::isinf()`: Comprueban si un número es NaN (Not a Number) o infinito.
* `std::rand()`, `std::srand()`: Generan números aleatorios. Aunque estas funciones son parte de la librería `cstdlib`,
  se utilizan comúnmente en operaciones matemáticas.

Estas funciones matemáticas son muy útiles para realizar cálculos numéricos y operaciones matemáticas en C++.

En resumen, las librerías `algorithm` y `math` son dos de las librerías estándar de C++ que proporcionan funciones y
algoritmos útiles para realizar operaciones matemáticas y de manipulación de datos en programas en C++. Estas
librerías son ampliamente utilizadas en la programación en C++ y ofrecen una amplia gama de funcionalidades para
realizar operaciones comunes en contenedores de datos y números.
